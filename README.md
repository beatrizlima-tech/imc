# 📌 Sobre o projeto

A **Calculadora de IMC** é uma aplicação web desenvolvida com **HTML**, **CSS** e **JavaScript** que calcula o Índice de Massa Corporal (IMC) a partir dos dados informados pelo usuário.

Após informar peso e altura, a aplicação realiza o cálculo automaticamente e apresenta a classificação correspondente de acordo com os parâmetros da Organização Mundial da Saúde (OMS).

Este projeto foi desenvolvido para praticar conceitos fundamentais do desenvolvimento Frontend, manipulação do DOM e lógica de programação.

# 📊 Classificação do IMC

| IMC            | Classificação   |
| -------------- | --------------- |
| Menor que 18,5 | Magreza         |
| 18,5 até 24,9  | Peso normal     |
| 25,0 até 29,9  | Sobrepeso       |
| 30,0 até 39,9  | Obesidade       |
| Acima de 40    | Obesidade grave |
